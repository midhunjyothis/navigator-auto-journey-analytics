# Navigator Auto Journey Analytics

## Objective
Design an end-to-end analytics framework that simulates Capital One Navigator’s digital car-buying journey:
Browse → Eligibility → Lead → Purchase.

## Architecture
Layered analytics architecture (Raw → Conformed → Analytics) with governed KPIs and experiment-ready data products.

## Data Model
Hybrid approach:
- Real-world vehicle & purchase data
- Simulated browsing, eligibility, personalization, and trigger behavior

## KPIs
Funnel conversion, personalization lift, eligibility effectiveness, trigger precision, and time-to-conversion.

## Data Quality & Governance
Business definitions, lineage, null/duplicate checks, anomaly detection, and metric ownership.

## Experimentation
A/B testing framework to measure incremental impact of personalization and pricing strategies.

## Dashboards
Self-service BI views designed for executives, product managers, and marketing partners.

## How to Run
Instructions to generate data, run transformations, and reproduce analyses.
